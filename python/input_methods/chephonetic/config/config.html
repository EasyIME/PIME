PIME 輸入法平台版本：


PIME 是在 Windows上使用 Python 3 開發 TSF 中文輸入法的平台，由作者 洪任諭(PCMan) 所開發，目前支援到 Windows 10，並且能支援 Windows app container (Metro mode)。可以讓開發者在不了解系統細節的情況下，單純使用 Python 就能實作支援 Windows 的輸入法模組。

注音是在 PIME 輸入法平台下，基於 CIN 碼表所實現的注音輸入法模組。本身移植了 PIME 平台下新酷音大部份的功能，並參考了微軟輸入法的一些操作方式，讓使用者轉移過來使用時，能夠馬上就上手，而不須要重新去適應。
